# Altair CAN バス通信プロトコル仕様書 (can_specification.md)

本ドキュメントは、Altair Module System で使用される各マイコンモジュール（MDD, Servo, Solenoid Valve）とPC（ROS2ブリッジ）間でやり取りされるCANフレーム（すべて **CAN1, ボーレート 1 Mbps**）のIDマップおよびデータ構造を規定する詳細仕様書です。

---

## 1. 総合 CAN ID マップ

本システムで使用される標準的なCAN ID割り当ての一覧です。

| CAN ID (Hex) | 方向 | モジュール | 内容 | ペイロード (DLC) |
|---|---|---|---|---|
| `0x100` | PC ➔ MCU | **Servo** | 6ch 角度指令 (各 1バイト) | 6バイト |
| `0x200` | PC ➔ MCU | **MDD** | Motor1 PIDパラメータ ＆ 車輪径 | 8バイト |
| `0x201` | PC ➔ MCU | **MDD** | Motor2 PIDパラメータ ＆ 車輪径 | 8バイト |
| `0x202` | PC ➔ MCU | **MDD** | Motor3 PIDパラメータ ＆ 車輪径 | 8バイト |
| `0x203` | PC ➔ MCU | **MDD** | Motor4 PIDパラメータ ＆ 車輪径 | 8バイト |
| `0x210` | PC ➔ MCU | **MDD** | 4ch 制御モード設定 (各 1バイト) | 4バイト |
| `0x220` | PC ➔ MCU | **MDD** | 4ch 目標値指令 (各 2バイト) | 8バイト |
| `0x230` | MCU ➔ PC | **MDD** | ステータス返信 (Limit SW, Error) | 8バイト (周期: 10ms) |
| `0x240` | MCU ➔ PC | **MDD** | 4ch エンコーダ現在角度 | 8バイト (周期: 10ms) |
| `0x250` | MCU ➔ PC | **MDD** | 4ch エンコーダ現在速度 | 8バイト (周期: 10ms) |
| `0x300` | PC ➔ MCU | **Solenoid** | 12ch 電磁弁 ON/OFF ビットマップ | 2バイト |

## 2. 各モジュールの詳細パケット仕様

### A. MDD (モータドライバモジュール)

#### ① PIDパラメータ設定 (`0x200` 〜 `0x203`)
- **DLC**: 8バイト
- **データ形式**: リトルエンディアン (16ビット符号付き整数 `int16_t` × 4)

| バイト | 型 | スケール・解釈 | 内容 |
|---|---|---|---|
| Byte 0-1 | `int16_t` | ゲイン × 1000 | **P ゲイン** (比例ゲイン) |
| Byte 2-3 | `int16_t` | ゲイン × 1000 | **I ゲイン** (積分ゲイン) |
| Byte 4-5 | `int16_t` | ゲイン × 1000 | **D ゲイン** (微分ゲイン) |
| Byte 6-7 | `int16_t` | 単位: [mm], 符号: 回転方向 | **車輪径 ＆ 出力方向**<br>絶対値 = 車輪径 [mm], 正=通常方向, 負=反転方向 |

#### ② 4ch 制御モード設定 (`0x210`)
- **DLC**: 4バイト
- **データ形式**: 8ビット符号無し整数 `uint8_t` × 4

| バイト | 型 | 設定値の解釈 | 内容 |
|---|---|---|---|
| Byte 0 | `uint8_t` | 0=速度, 1=角度, 2=位置 | Motor 1 制御モード |
| Byte 1 | `uint8_t` | 0=速度, 1=角度, 2=位置 | Motor 2 制御モード |
| Byte 2 | `uint8_t` | 0=速度, 1=角度, 2=位置 | Motor 3 制御モード |
| Byte 3 | `uint8_t` | 0=速度, 1=角度, 2=位置 | Motor 4 制御モード |

#### ③ 4ch 目標値指令 (`0x220`)
- **DLC**: 8バイト
- **データ形式**: リトルエンディアン `int16_t` × 4 (制御実行モードでのみ有効)

| バイト | 型 | スケール・解釈 | 内容 |
|---|---|---|---|
| Byte 0-1 | `int16_t` | 物理値 × 10 | Motor 1 目標値 (速度:rps, 角度:deg, 位置:mm) |
| Byte 2-3 | `int16_t` | 物理値 × 10 | Motor 2 目標値 (速度:rps, 角度:deg, 位置:mm) |
| Byte 4-5 | `int16_t` | 物理値 × 10 | Motor 3 目標値 (速度:rps, 角度:deg, 位置:mm) |
| Byte 6-7 | `int16_t` | 物理値 × 10 | Motor 4 目標値 (速度:rps, 角度:deg, 位置:mm) |

#### ④ ステータス返信 (`0x230`)
- **DLC**: 8バイト
- **配信周期**: 10ms

| バイト | 型 | 設定値の解釈 | 内容 |
|---|---|---|---|
| Byte 0 | `uint8_t` | 0=OFF (開放), 1=ON (押下) | Limit SW 1 状態 |
| Byte 1 | `uint8_t` | 0=OFF (開放), 1=ON (押下) | Limit SW 2 状態 |
| Byte 2 | `uint8_t` | 0=OFF (開放), 1=ON (押下) | Limit SW 3 状態 |
| Byte 3 | `uint8_t` | 0=OFF (開放), 1=ON (押下) | Limit SW 4 状態 |
| Byte 4 | `uint8_t` | ビットフラグ (下表参照) | システムエラーコード |
| Byte 5 | `uint8_t` | 0=パラメータ設定中, 1=制御実行中 | システム動作モード状態 |
| Byte 6-7 | `uint8_t` | 固定値 0 | 未使用領域 |

**エラーコード（Byte 4）のビットフラグ解釈:**
- `0x00`: NORMAL (正常動作)
- `0x01`: INIT_TIMEOUT (エンコーダやモータ初期化時のタイムアウト)
- `0x02`: CAN_RX_TIMEOUT (一定時間指令の受信無しによる安全強制停止)
- `0x04`: CAN_TX_FAIL (フィードバック送信失敗)

#### ⑤ エンコーダ角度返信 (`0x240`)
- **DLC**: 8バイト
- **データ形式**: リトルエンディアン `int16_t` × 4 (常時10ms周期配信)

| バイト | 型 | スケール・解釈 | 内容 |
|---|---|---|---|
| Byte 0-1 | `int16_t` | 生データ 0.1 deg 単位 | Motor 1 現在角度 (例: 1234 ➔ 123.4度) |
| Byte 2-3 | `int16_t` | 生データ 0.1 deg 単位 | Motor 2 現在角度 |
| Byte 4-5 | `int16_t` | 生データ 0.1 deg 単位 | Motor 3 現在角度 |
| Byte 6-7 | `int16_t` | 生データ 0.1 deg 単位 | Motor 4 現在角度 |

#### ⑥ エンコーダ回転速度返信 (`0x250`)
- **DLC**: 8バイト
- **データ形式**: リトルエンディアン `int16_t` × 4 (常時10ms周期配信)

| バイト | 型 | スケール・解釈 | 内容 |
|---|---|---|---|
| Byte 0-1 | `int16_t` | 生データ 0.01 rps 単位 | Motor 1 現在速度 (例: 123 ➔ 1.23 rps) |
| Byte 2-3 | `int16_t` | 生データ 0.01 rps 単位 | Motor 2 現在速度 |
| Byte 4-5 | `int16_t` | 生データ 0.01 rps 単位 | Motor 3 現在速度 |
| Byte 6-7 | `int16_t` | 生データ 0.01 rps 単位 | Motor 4 現在速度 |

### B. Servo (6ch サーボモジュール)

#### 角度指令 (`0x100`)
- **DLC**: 6バイト
- **データ形式**: 1バイト(uint8_t) × 6ch

| バイト | 型 | スケール・解釈 | 内容 |
|---|---|---|---|
| Byte 0 | `uint8_t` | 0 〜 180 度 | Servo 1 目標角度 |
| Byte 1 | `uint8_t` | 0 〜 180 度 | Servo 2 目標角度 |
| Byte 2 | `uint8_t` | 0 〜 180 度 | Servo 3 目標角度 |
| Byte 3 | `uint8_t` | 0 〜 180 度 | Servo 4 目標角度 |
| Byte 4 | `uint8_t` | 0 〜 180 度 | Servo 5 目標角度 |
| Byte 5 | `uint8_t` | 0 〜 180 度 | Servo 6 目標角度 |

---

### C. Solenoid Valve (12ch 電磁弁モジュール)

#### 12ch ON/OFF指令 (`0x300`)
- **DLC**: 2バイト
- **データ形式**: ビットマップ (各ビット 1=ON/開く, 0=OFF/閉じる)

| バイト | ビット | 対応する物理出力ピン | 内容 |
|---|---|---|---|
| **Byte 0** | bit 0 | PA0 | Valve 1 コントロール |
| | bit 1 | PA1 | Valve 2 コントロール |
| | bit 2 | PA6 | Valve 3 コントロール |
| | bit 3 | PA7 | Valve 4 コントロール |
| | bit 4 | PA8 | Valve 5 コントロール |
| | bit 5 | PA9 | Valve 6 コントロール |
| | bit 6 | PA15 | Valve 7 コントロール |
| | bit 7 | PB3 | Valve 8 コントロール |
| **Byte 1** | bit 0 | PB6 | Valve 9 コントロール |
| | bit 1 | PB7 | Valve 10 コントロール |
| | bit 2 | PB8 | Valve 11 コントロール |
| | bit 3 | PB9 | Valve 12 コントロール |
| | bit 4-7 | - | 未使用 (0固定) |

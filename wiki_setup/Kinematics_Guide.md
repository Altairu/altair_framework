本ドキュメントは、Altair Framework の Web UI（Blockly ビジュアル automation）で生成される**ロボット足回りの運動学制御**について、数式・パラメータ・実装詳細をまとめた技術仕様書です。

---

## 座標系と基本パラメータ

### 座標系定義

```
      +Y (前方)
       ↑
       |  Ω (旋回角速度)
       |  ⤴ (反時計回り=正)
    ←─+─→ +X (右方)
       |
```

### 基本パラメータ

| パラメータ | 記号 | 単位 | 説明 |
|:---|:---:|:---:|---|
| **前後速度** | $V_y$ | mm/s | 正=前進、負=後退 |
| **左右速度** | $V_x$ | mm/s | 正=右移動、負=左移動 |
| **旋回角速度** | $\Omega$ | rad/s | 正=反時計回り（左旋回） |
| **旋回直径** | $L$ | mm | ロボット中心回りの旋回半径パラメータ |
| **車輪径** | $d$ | mm | 各モータのホイール直径 |

## 4つのロボット構成タイプ

### 1. 3輪オムニ (3_OMNI)

**ホイール配置：** 120°間隔で3つのオムニホイール

```
          M1 (前)
           △
          /|\
         / | \
        /  |  \
       /   |   \
    M2○────┼────○M3
      120° | 120°
```

#### 運動学式

$$v_1 = V_y + r_t \cdot \Omega$$

$$v_2 = -0.866025 \times V_x - 0.5 \times V_y + r_t \cdot \Omega$$

$$v_3 = 0.866025 \times V_x - 0.5 \times V_y + r_t \cdot \Omega$$

ここで $r_t = \frac{L}{2000}$

#### 特徴
- 全方向移動可能  
- 最小回転半径が小さい  
- シンプルな構成

---

### 2. 4輪オムニ (4_OMNI)

**ホイール配置：** 45°間隔で4つのオムニホイール

```
    LF ◇─────────◇ RF
      ╱           ╲
     ╱             ╲
    │      +Y       │
    │       ↑       │
 ───┼───────+───────┼──→ +X
    │               │
     ╲             ╱
      ╲           ╱
    LR ◇─────────◇ RR
```

#### 運動学式

$$v_{LF} = (-V_x + V_y - r_t \cdot \Omega) \times \frac{\sqrt{2}}{2}$$

$$v_{RF} = (V_x + V_y + r_t \cdot \Omega) \times \frac{\sqrt{2}}{2}$$

$$v_{LR} = (-V_x - V_y + r_t \cdot \Omega) \times \frac{\sqrt{2}}{2}$$

$$v_{RR} = (V_x - V_y - r_t \cdot \Omega) \times \frac{\sqrt{2}}{2}$$

#### 特徴
- 全方向移動・横スライド・その場旋回  
- 4輪構成で安定性高い  
- 45°オムニホイール標準構成

---

### 3. 4輪メカナム (4_MECANUM)

**ホイール配置：** ローラー付きメカナムホイール（正方形配置）

```
    LF □─────────□ RF
      │    MDD    │
      │    M1-M4  │
      │           │
    LR □─────────□ RR
```

#### 運動学式

$$v_{LF} = V_y - V_x - r_t \cdot \Omega$$

$$v_{RF} = V_y + V_x + r_t \cdot \Omega$$

$$v_{LR} = V_y + V_x - r_t \cdot \Omega$$

$$v_{RR} = V_y - V_x + r_t \cdot \Omega$$

#### 特徴
- 最も一般的な全方向移動構成  
- 全方向速度を実現  
- 計算がシンプル

---

### 4. 差動2輪 (2_DIFF)

**ホイール配置：** 左右 2 個のホイールグループ（タンク型）

```
    L ◯─────────◯ R
      │ (中心)  │
    L ◯─────────◯ R
```

#### 運動学式

$$v_L = V_y - r_t \cdot \Omega$$

$$v_R = V_y + r_t \cdot \Omega$$

左右各グループのモータは同じ速度を受け取ります。

#### 特徴
- 最もシンプルな制御  
- 前後移動と旋回のみ  
- 信頼性が高い

## パラメータ設定

### 各モータの個別設定

#### 制御モード

```
Mode: [Speed ▼] [Angle ▼] [Position ▼]
```

- **Speed**: 目標値を rps（回転数/秒）として指定
- **Angle**: 目標値を度（deg）として指定
- **Position**: 目標値をミリメートル（mm）として指定

#### 車輪径 (Diameter)

```
Diameter: [100.0] mm
```

ロボットの実際のホイール径を測定して入力してください。

#### 回転方向（Direction）

```
Direction: [+1 ▼] [-1 ▼]
```

- **+1**: 正の目標値で正回転（通常）
- **-1**: 正の目標値で逆回転（反転）
